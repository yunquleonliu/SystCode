# SystCode-

A LeetCode style repository for OS/Memory/C system level learning and practice.

## Overview

This repository provides comprehensive resources for learning operating system concepts, memory management, and system-level programming in C. It's designed to help you prepare for technical interviews and deepen your understanding of low-level system concepts.

## Contents

### 📚 [OS Level System Design Review](OS-System-Design-Review.md)

A comprehensive list of OS level system design topics including:
- Memory Management
- Process Management
- File Systems
- I/O Systems
- Concurrency and Synchronization
- CPU Scheduling
- Deadlocks
- Virtual Memory
- Networking
- Security
- Distributed Systems
- Real-Time Systems

Each topic includes key concepts, questions to consider, and practical insights.
